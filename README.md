# GrandLog

GrandLog is a vehicle expense, fuel, and maintenance tracking API built with FastAPI.

## Features

- Vehicle CRUD API
- Maintenance tracking
- Fuel log tracking
- Fuel economy analytics
- SQLite database
- Layered backend architecture
- Auto-generated API documentation

## Tech Stack

- Python
- FastAPI
- SQLAlchemy
- SQLite
- Pydantic
- Uvicorn

## Project Structure
